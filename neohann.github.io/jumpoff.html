Poverty and Heart disease mortality rate in United States

According to this article poverty is both a cause and consequence of poor health.

Is there a relationship between the poverty and heart disease mortality rate? Do other factors such as location or weather contribute to the high mortaility rate?

The project follows the Martini Glass narrative visualization structure. This page represents the "jumping off point" of the visualization structure. The user has experienced author driven contents and is now ready for the reader driven content. Final page contains reader driven content and it allows users to interact with data and draw their own conclusions.

User Interface Events

The buttons in the navigation bar and the Prev, Next button at the bottom of this page enable the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this story.

Annotations

There are no annotations or visual charts on this page. However, all other sections of this visualization story has annotated visuals.

Triggers

The mouse click acts as a trigger event.

Parameters

This section is hyper linked to the previous and next section. The name of the target section is passed as an iput paramter to the buttons and navgiation bar items.
